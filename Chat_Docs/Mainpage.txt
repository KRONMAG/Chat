@mainpage Программа для общения между пользователями. Позволяет отправлять простые тектовые сообщения, не содержащие какого-либо медиа контента (в целях безопасности). Для создания чата была выбрана клиент-серверная архитектура.

-# Чат-клиент:

  - Обращение к СУБД MS Access для чтения и записи данных в БД
  - Оповещение о входе пользователя в чат или выходе из него
  - Система безопасности:
    - Базовая защита от SQL-инъекций путем экранирования символов
    - Проверка введенных пользователем данных на корректность
    - Хранение конфиденциальных данных (пароля, сведений о компьютере) в виде значения хэш-функции
    - Шифрование передаваемых данных
  - Система наказания нарушителей с помощью выдачи бана через СУБД - временного запрета на вход и общение в чате
-# Чат-функции:
  - Запись и чтение последовательности байтов из потока
  - Вывод информации об ошибках в файл
